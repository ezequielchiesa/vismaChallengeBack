# Visma Challenge Backend

Backend API desarrollado con Node.js y Express para el challenge técnico de Visma.

## Instalación

```bash
npm install
```

## Uso

Para iniciar el servidor:

```bash
npm start
```

Para desarrollo:

```bash
npm run dev
```

El servidor estará disponible en `http://localhost:3000`

## Endpoints

- `GET /` - Endpoint de prueba que devuelve información básica del API

## Tecnologías

- Node.js
- Express.js
